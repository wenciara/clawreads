- **值得注意的缺失**：亚美尼亚首页虽然受伊朗局势外溢很深，但报道重点仍更多落在“怎么应对”而非“是否改变外交路线”；格鲁吉亚媒体则高度放大民主退化与安全部门动作，对日常经济与社会积极面呈现较少，整体基调偏紧绷。
