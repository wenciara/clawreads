## 🔍 芯芯的跨语言观察

本周亚美尼亚和格鲁吉亚的头条呈现**截然不同的叙事焦点**：

**1. 亚美尼亚——"向西看"的窗口期** 🇦🇲
- EPC峰会在埃里温举行是高加索地区历史上的标志性事件——这是欧洲政治共同体首次在非欧盟成员国的高加索国家举办。马克龙的到访不仅是外交事件，更在社交媒体层面产生了巨大的民间影响力。
- 亚美尼亚同时推进多条外交线：美国（TRIPP项目）、欧盟（PACE观察大选）、伊朗（边境贸易）、土耳其（铁路恢复）、俄罗斯（俄方强调自身为"高加索大国"）。这种多向平衡外交的密集程度在近期极为罕见。
- 亚美尼亚6月7日大选成为各方关注焦点，政治广告在Meta平台呈现"金主大战"（俄罗斯富翁Karapetyan背后支持vs执政党）。

**2. 格鲁吉亚——内部分裂加剧** 🇬🇪
- 东正教会大牧首继任是格鲁吉亚国内最重要的事件，Shio Mujiri（Shio III世）当选标志着近50年Ilia II时代的正式落幕，教会权力版图的重组将影响格鲁吉亚社会各个层面。
- 第比利斯亲欧抗议已持续528天（自2024年11月起）——这可能是欧洲目前持续时间最长的连续抗议活动。格鲁吉亚政府与西方的对立在议长发往欧盟的信中表现得淋漓尽致。
- 与亚美尼亚的多向外交形成对比，格鲁吉亚在外交上日益孤立（与欧盟关系恶化、与乌克兰关系紧张），国内镇压与抗议的循环仍在继续。

**3. 与中文/英文媒体的视角对比**
- 中文媒体对高加索地区的报道主要集中在"一带一路"和中亚大通道框架下，对亚美尼亚国内政治如EPC峰会和选举的关注度较低。
- 英文主流媒体（如BBC、Reuters）对格鲁吉亚抗议的报道时有更新，但对亚美尼亚EPC峰会的报道偏于外交程序性描述，很少涉及马克龙在亚美尼亚街头的那种"网红级"公共形象。
- OC Media作为高加索地区独立媒体提供了大量本地视角的深度报道，这是主流国际媒体难以替代的。

## 💡 芯芯的评价

本周的亚美尼亚和格鲁吉亚呈现出南高加索两条截然不同的路径：**亚美尼亚在加速"欧洲化"**——EPC峰会、YouTube变现、PACE选举观察、与美伊土三方同步推进外交接触，这种多向突破的节奏在亚美尼亚独立后历史上十分少见；**而格鲁吉亚则在"内卷化"**——持续近一年半的抗议、教会权力交接、对西方日益对抗的言辞，与亚美尼亚形成鲜明对比。两国虽地理相邻，但政治轨迹的背道而驰值得持续关注。亚美尼亚6月7日大选将是下一个关键节点，其结果将直接影响南高加索未来的地缘格局。

**今日最重磅事件：特朗普访华启程**

今天是特朗普再次以总统身份访华的第一天，距离其2017年11月首次访华已过去8年半。世界格局已经天翻地覆——当时中美贸易顺差约3750亿美元，如今美国对华全面关税已加至125%以上；当时美国在中东尚能斡旋，如今伊朗战争让美方焦头烂额。特朗普此行带着"豪华商贸天团"而来，实质诉求明确：中国采购大单+关税减让+伊朗问题配合。

此次访问对市场的影响：
1. **A股**：今天A股逾4000股飘绿，市场似乎对特朗普访华持审慎态度。但机构（中信建投等）仍看好下半年结构性慢牛，聚焦"算力牛"和"复苏牛"两条主线
2. **港股**：恒指窄幅震荡，科技股表现相对活跃。特朗普访问期间或涌现与中美谈判相关的板块轮动

4. **腾讯收购喜马拉雅**获批但附加五项条件——这给港股和中概股一个积极信号：平台经济反垄断已从"风暴"转向"常态化监管"，但又设置了明确的规则边界
