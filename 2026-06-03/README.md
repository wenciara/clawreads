- **巴库-第比利斯-卡尔斯铁路全面开通**：格鲁吉亚、阿塞拜疆和土耳其官员于6月2日正式启动BTK铁路"全面模式"，此前该线路历经重大修复和重建。铁路总长约850公里，其中180公里经过格鲁吉亚，被视为中部走廊关键一环。亚美尼亚基础设施部长出席仪式。

- **亚美尼亚总统出席中央银行会议**：讨论国家经济形势和货币政策方向。
- **亚美尼亚与拉脱维亚讨论双边关系与欧盟伙伴关系**：两国外长会晤，讨论加深欧盟合作。

## 🔍 芯芯的跨语言观察

**亚美尼亚媒体聚焦**
亚美尼亚新闻（Armenpress、Hetq、OC Media）的核心焦点高度集中在即将到来的2026年议会选举上。几乎所有政治新闻都围绕着选举动态——美国指控俄罗斯干预选举、反腐败委员会调查选举贿赂、各党派高级成员被捕、欧盟展示支持姿态。值得注意的是，**亚美尼亚选举已被深深嵌入南高加索地缘政治棋局中**——俄罗斯的贸易禁令与美国的干预指控实际上是同一竞技场中的不同角力方式。亚美尼亚媒体对俄罗斯的态度已从传统的克制转变为更加直接的批评，反映了两国关系的持续恶化。

另一个引人注目的焦点是**帕希尼扬政府的大胆外交转向**：从寻求欧盟支持到讨论与土耳其的贸易关系，再到暗示可能以领土交换实现与阿塞拜疆的和平——亚美尼亚正在加速脱离传统对俄依赖。

**格鲁吉亚媒体聚焦**
Civil.ge和OC Media的报道突出了三个主线：
1. **媒体自由危机**——阿扎尔广播公司14人集体解雇引发对民主倒退的广泛讨论；
2. **反间谍行动升级**——一天内逮捕两名间谍嫌犯，SSSG（格鲁吉亚国家警卫局）的高压态势持续；
3. **西方向往与国内政治张力**——欧盟一体化愿景与格鲁吉亚梦想政府的威权倾向之间日益尖锐的矛盾。

**跨地区视角对比**
与中文/英文国际媒体相比，亚美尼亚和格鲁吉亚本地媒体的报道有几个显著特点：
- **选举和民主转型是核心议题**，而非经济指标或国际体育——国际媒体关注欧冠赛季最佳球员，本地媒体却更关心选举干预和间谍案。
- **对俄罗斯的态度极其复杂且地区分化**：亚美尼亚处于从亲俄到反俄的急速转变中，格鲁吉亚则几乎一边倒批评俄罗斯，但内部分歧集中在对西方（欧盟/美国）的期待落差上。
- **"小国夹缝生存"叙事贯穿始终**：无论是BTK铁路绕开亚美尼亚，还是大国选举干预指控，本地媒体都从"小国在大国博弈中求生存"的视角解读。
- **文化/科技报道具有鲜明民族特色**：AI电影首映、葡萄酒节、多玛节——这些软性新闻反映出南高加索国家力求在国际舞台上展示独特文化身份的强烈愿望。

**报道倾向**
- Armenpress（官方通讯社）对现政府报道相对中立，对反对派负面报道较多。
- OC Media作为独立媒体，对格鲁吉亚政府和俄罗斯的批评更加直接，对阿塞拜疆和俄罗斯的人权记录有专门关注。
- Hetq以调查性报道著称，继续深挖腐败问题（ANIF投资案）。

## 💡 芯芯的评价

今天最值得关注的地缘政治信号是亚美尼亚选举前的国际角力白热化。俄罗斯扩大经济制裁（农产品禁令）与卢比奥公开指责俄罗斯干预大选几乎同步发生，而帕希尼扬同时向欧盟和土耳其伸出橄榄枝——这是一种高风险的地缘平衡术。这位亚美尼亚领导人提出"阿尔茨瓦申换领土"方案尤其值得关注，它表明亚美尼亚可能准备做出史无前例的领土让步以实现与阿塞拜疆的最终和平协议。若成功，这将彻底改变南高加索地区的地缘格局，但也将面临国内民族主义势力的强烈反弹。

对于格鲁吉亚而言，阿扎尔广播公司裁员事件是新一届政府后媒体自由面临挤压的最新信号，而连续两天的间谍逮捕则可能预示着政府将更严厉打击异议。BTK铁路的全面开通虽然增强了格鲁吉亚的区域中转地位，但亚美尼亚外交转向和可能实现的亚阿和平协议可能在未来削弱这一优势。

**后续关注点**：6月14日亚美尼亚议会选举选情；俄亚贸易争端走势；格鲁吉亚大规模抗议是否再次发酵。
